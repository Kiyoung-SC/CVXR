# CVXR

A collection of convex optimization practice problems using CVXR in R.

## Practice Problems

This repository contains practice problems for learning convex optimization with CVXR. Visit the [`practices/`](practices/) directory to find:

- **Linear Programming** - Production planning and resource allocation
- **Quadratic Programming** - Portfolio optimization and risk management  
- **Least Squares** - Regression and curve fitting

Each practice includes detailed comments, example code, and real-world applications.

## Getting Started

1. Install CVXR:
   ```r
   install.packages("CVXR")
   ```

2. Explore the practice problems in the `practices/` directory
3. Run the examples and modify them to learn
